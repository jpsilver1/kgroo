## 2. 화면별 구조 분석

### 2-1. 홈 (Dashboard)
- 상단: 시장 지수 바 (코스피 / 코스닥 / 나스닥 / 다우존스 / S&P500)
- 중단: 피드 섹션 (전체 / 관심 / 보유 탭)
  - 기업설명회, 분기보고서, 대량보유자변동보고 카드
- 하단: 인기 검색기업 테이블 (순위 / 기업명 / 현재가 / 변동률 / 시가총액)
- 하단: 배당소득 분리과세 수혜기업 테이블

### 2-2. 스크리너 (Screener)
- 좌측: 카테고리 필터 패널
  - 내탑픽 / 펀더멘탈 / 시장 거래소 / 성장/수익지표 / 실적 시나리오 / 수주잔고 등
- 우측 상단: 탭 (전체기업 / 관심기업 / 보유기업 / 주요지수)
- 우측 본문: 종목 테이블
  - 컬럼: 기업명 / 현재가 / 변동률 / 시가총액
  - 검색된 기업 수 표시
  - 국내 / 미국 전환 가능

### 2-3. 포트폴리오 (Portfolio)
- 상단: 계좌 요약 카드
  - 전체계좌 / 계좌번호 선택
  - 평가손익 / 실현손익 / 평가금액 / 예수금
- 하단: 자산현황 / 보유종목 탭
  - 테이블 컬럼: 종목명 / 현재가 / 비중 / 평가수익률 / 보유수량 / 평가금액 / 매입금액 / 배당예상금액 / 매수가 / 평가손익수익률 / 시가배당수익률

### 2-4. 공시
- 테이블: 날짜 / 시간 / 기업명 / 보고서명 / 제출인
- 탭: 공시목록 / 납입자본 / 내부자거래 / 공시알림설정 / 사업부문 / 공공계약 / 임원공시 / 사업권

### 2-5. IR
- 테이블: 수행일 / 시간 / 기업명 / 내용 / 대상자 / 방법 / 담당자 / 전화번호 / 보고일 / 파일다운 / 공시

### 2-6. 증권사리포트
- 테이블: 기업명 / 기업명 / 목적 / 제목 / 목표주가 / 현재가 / 목표주가수익률 / 증권사 / 애널리스트 / RATING

### 2-7. 뉴스
- 카드형 레이아웃
- 썸네일 이미지 + 기업명 태그 + 제목 + 출처 + 시간

### 2-8. 버틀러사용법
- 유튜브 썸네일 카드 그리드
- 카테고리: 튜토리얼 / 소개영상

### 2-9. 구독플랜
- 연간구독: 월 19,900원
- 월간구독: 월 24,900원
- 무료 vs 구독 기능 비교표
- 차트 미리보기 이미지 6종
- FAQ 아코디언

## 4. 디자인 시스템

### 컬러
```
배경색:        흰색 (#ffffff)
패널색:        연회색 (color-base-200)
포인트 컬러:   #664efa (보라/파랑)
보조 컬러:     #7468ff
테두리:        var(--color-slate-200)
상승(빨강):    color-error 계열
하락(파랑):    color-info 계열
성공(초록):    color-success 계열
```

### 폰트
```
메인: Pretendard
fallback: sans-serif
```

### Border Radius
```
버튼/박스: 0.5rem (8px)
인풋:      0.25rem (4px)
```

### 레이아웃 수치
```
사이드바: 약 40px (아이콘만) ~ 220px (펼침)
헤더 높이: 약 48px
```

## 5. UX 특징

- 정보 밀도 높지만 복잡하지 않음
- 테이블 중심 레이아웃 (주식 데이터 특성상)
- 우측 패널 슬라이드 방식으로 기업 검색
- 탭으로 국내/미국 전환
- 모바일에서 일부 요소 숨김 처리 (mobileOnlyFlex)
- 상단 고정 네비게이션

## 6. 버틀러의 한계 (내주가 차별화 포인트)

| 버틀러 한계 | 내주가 해결책 |
|------------|-------------|
| 기업분석 중심, 자산관리 기능 약함 | 개인 자산 포트폴리오 중심 |
| AI 분석 기능 없음 | AI 맞춤 자산 조언 |
| 유료 구독 부담 (월 19,900~24,900원) | 저비용 서민형 요금제 |
| 자산군 비중 설정 기능 없음 | 현금/주식/금/달러 비중 설정 |
| 자동매매 기능 없음 | 단계적 자동매매 알림 |
