## 프롬프트

```
내주가 앱의 Dashboard 화면을 구현해줘.
01_layout.md로 만든 레이아웃 안에 들어갈 Dashboard.jsx 내용이야.

## 화면 구성 (위에서 아래 순서)

### 1. 시장 지수 바 (상단)
- 코스피 / 코스닥 / 나스닥 / S&P500 / 다우존스
- 각각: 지수명 / 현재값 / 변동률 (상승 초록 #22c55e / 하락 빨강 #ef4444)
- 가로 스크롤 가능한 카드 형태

### 2. 메인 2단 그리드 (중단)
좌측 (60%):
- 내 자산 비중 도넛차트
  - 항목: 현금 / 주식 / 금 / 달러 / 기타
  - 우측에 항목별 비중 % 표시
  - Recharts PieChart 사용

우측 (40%):
- AI 오늘의 한줄 조언 카드
  - 아이콘 + 텍스트
  - 배경: #1a1a24
  - 포인트 컬러 테두리
- 자산 요약 카드
  - 총 자산 / 오늘 수익 / 수익률

### 3. 인기 검색기업 TOP 10 테이블 (하단)
컬럼: 순위 / 기업명 / 현재가 / 변동률 / 시가총액
- 변동률: 상승 초록, 하락 빨강
- hover 시 행 배경색 변경
- 우측 상단: 국내 / 미국 탭 전환

## 더미 데이터 포함
- 시장 지수 5개
- 자산 비중 5개 항목
- 인기기업 10개

## 조건
- Recharts 라이브러리 사용 (차트)
- 모든 수치 더미데이터로 작성
- 카드 컴포넌트 분리 권장
- 테이블 hover 효과 포함
```
