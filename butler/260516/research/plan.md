# 내주가 (NAE JUGA) 서비스 기획
> 작성일: 2026-05-15
> 참고: research.md (버틀러 벤치마킹)
> 목표: 서민형 AI 자산관리 대시보드

---

## 1. 서비스 개요

- **서비스명**: 내주가 (NAE JUGA)
- **핵심 포지셔닝**: AI가 개인 상황에 맞춰 자산 포트폴리오를 설계하고 관리해주는 서민형 금융 에이전트
- **타겟**: 20~40대 직장인, 사회초년생, 소액투자자, 자영업자
- **차별화**: 버틀러(기업분석 중심) vs 내주가(AI 자산관리 중심)

---

## 2. 기술 스택

### Frontend
```
- Framework: React + Vite
- Styling: Tailwind CSS
- 차트: Recharts
- 애니메이션: Framer Motion
- 아이콘: Lucide React
```

### Backend
```
- Framework: Python FastAPI
- 주식 데이터: FinanceDataReader (무료)
- 실시간 데이터: 한국투자증권 KIS API (추후)
- AI: Anthropic Claude API
```

### Infra
```
- Frontend 배포: Vercel
- Backend 배포: Railway
- DB: Supabase (PostgreSQL)
- 버전관리: GitHub
```

## 5. 화면별 기능 명세

### 5-1. Dashboard (1순위)
```
상단바:
- 코스피 / 코스닥 / 나스닥 / S&P500 실시간 지수

메인 콘텐츠:
- 내 자산 비중 도넛차트 (현금/주식/금/달러/기타)
- 오늘의 AI 한줄 조언
- 인기 검색기업 TOP 10 테이블
  (순위/기업명/현재가/변동률/시가총액)

사이드:
- 최근 공시 알림
- 관심종목 변동률
```

### 5-2. Portfolio (1순위)
```
상단 요약 카드:
- 총 자산 / 평가손익 / 수익률

자산 배분 섹션:
- 현금 / 예금 / 주식 / 금 / 달러 / 보험 / 부동산
- 현재 비중 vs 목표 비중 비교 차트

보유종목 테이블:
- 종목명 / 현재가 / 비중 / 수익률 / 보유수량
- 국내 / 미국 탭 전환
```

### 5-3. Screener (2순위)
```
좌측 필터 패널:
- 시장 (국내/미국)
- 시가총액 구간
- 업종
- PER / PBR / ROE 범위
- 배당수익률

우측 결과 테이블:
- 기업명 / 현재가 / 변동률 / 시가총액
- 관심기업 추가 버튼
```

### 5-4. AI 자산조언 (3순위)
```
입력:
- 나이 / 직업 / 월소득 / 투자성향
- 현재 자산 구성

출력:
- AI 맞춤 포트폴리오 제안
  (현금 X% / 주식 X% / 금 X% / 달러 X%)
- 추천 이유 설명
- 시장 상황 요약
```
