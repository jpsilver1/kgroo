# 내주가 MVP 기능 명세

## 메뉴 구조
버틀러 → 내주가
홈 → Dashboard
스크리너 → Screener
관심기업 → Watchlist
포트폴리오 → Portfolio
공시/뉴스 → News & Reports
내메모 → Memo
(신규) → AI 자산 조언
(신규) → 자산 비중 설정

## MVP 우선순위
1순위: Dashboard + Portfolio (자산 비중 시각화)
2순위: Screener + Watchlist
3순위: AI 분석 + News 요약
4순위: 자동매매 알림

## 첫 화면에 반드시 보여야 할 것
1. 내 전체 자산 현황 (원형 차트)
2. 자산군별 비중 (현금/주식/금/달러)
3. 오늘의 수익률
4. AI 한줄 조언
5. 관심 종목 변동률
