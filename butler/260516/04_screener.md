## 프롬프트

```
내주가 앱의 Screener 화면을 구현해줘.
01_layout.md로 만든 레이아웃 안에 들어갈 Screener.jsx 내용이야.

## 화면 구성

### 좌측 필터 패널 (너비 240px, 고정)
필터 카테고리:
- 시장: 국내 / 미국 라디오 버튼
- 시가총액: 슬라이더 (0 ~ 100조)
- 업종: 드롭다운 (전체 / 반도체 / 바이오 / 금융 / 에너지 등)
- PER: 입력 범위 (최소 ~ 최대)
- PBR: 입력 범위
- ROE: 입력 범위
- 배당수익률: 입력 범위

하단:
- 필터 초기화 버튼
- 검색 적용 버튼 (포인트 컬러)

### 우측 결과 영역
상단:
- "검색된 기업 N개" 텍스트
- 정렬 기준 드롭다운 (시가총액순 / 변동률순)

테이블:
- 컬럼: 순위 / 기업명 / 현재가 / 변동률 / 시가총액
- 변동률: 상승 초록 / 하락 빨강
- 행 hover 효과
- 행 클릭 시 관심기업 추가 버튼 노출
- 페이지네이션 (20개씩)

## 조건
- 필터 상태 React useState로 관리
- 더미 데이터 50개 이상
- 필터 적용 시 목록 즉시 업데이트
- 로딩 스피너 (필터 적용 시 0.5초)
```
