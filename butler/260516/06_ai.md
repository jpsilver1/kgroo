## 프롬프트

```
내주가 앱의 AI 자산조언 기능을 구현해줘.
백엔드(FastAPI) + 프론트엔드(React) 둘 다 작업해줘.

## 백엔드 (FastAPI)

### API 추가
POST /api/ai/advice
Body:
{
  "age": 30,
  "job": "직장인",
  "monthly_income": 4000000,
  "investment_style": "중립형",
  "current_assets": {
    "cash": 25,
    "stock": 45,
    "gold": 10,
    "dollar": 15,
    "other": 5
  }
}

응답:
{
  "advice": "AI 조언 텍스트",
  "recommended_portfolio": {
    "cash": 30,
    "stock": 40,
    "gold": 10,
    "dollar": 15,
    "other": 5
  },
  "reason": "추천 이유"
}

### Claude API 연동
- Anthropic SDK 사용
- 시스템 프롬프트: 금융 자산관리 전문가 역할
- 사용자 정보 기반 맞춤 포트폴리오 추천
- 응답은 반드시 JSON 형태로

## 프론트엔드 (React)

### AIAdvice.jsx 화면 구성
1. 입력 폼
   - 나이 (숫자 입력)
   - 직업 (드롭다운: 직장인/자영업자/학생/주부/기타)
   - 월 소득 (숫자 입력)
   - 투자성향 (라디오: 안정형/중립형/공격형)
   - 현재 자산 비중 (슬라이더 5개, 합계 100%)

2. AI 분석 버튼 (포인트 컬러)
   - 클릭 시 로딩 애니메이션

3. 결과 카드
   - AI 조언 텍스트
   - 현재 vs 추천 포트폴리오 비교 차트
   - 추천 이유

## 조건
- .env에 ANTHROPIC_API_KEY 추가
- 로딩 중 스피너 표시
- 에러 시 메시지 표시
```
